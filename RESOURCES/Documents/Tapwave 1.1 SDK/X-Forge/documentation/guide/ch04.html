Introduction

Artists should read the 'X-Forge for Artists' chapter of this guide before starting to create content for the X-Forge Game Engine.

X-Forge art toolchain requires the following software:

  • Discreet 3d studio max 5.0 or 5.1 (4.0 and 4.2 are supported by separate request)
  • Some 2D graphic tools eg. Adobe Photoshop
  • Exporter for 3d studio max. There are two optional exporters available: X-Forge Exporter (included in /tools/exporter/) (for X-Forge Core only users) MaxFlex Flexporter plugin for 3d studio max (included in /tools/maxflex/) (for X-Forge full edition users)

See exporter documentation about installing it. More information about using different exporters can be found in the 'X-Forge for Artists' chapter.
