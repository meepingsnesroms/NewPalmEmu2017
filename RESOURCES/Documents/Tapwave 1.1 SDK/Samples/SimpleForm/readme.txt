This example demonstrates how to properly set-up an application's form using the
Pin Input Manager.  From this example you can easily modify the various PINs API calls
to change the screen orientation, grafitti area visibility, status bar visibility, and 
trigger button states.

The important functions to examine are MainFormHandleEvent and MainFormResize.

------------------------------------------------------------
Build Instructions
------------------------------------------------------------

The SimpleForm code is compiled using Metrowerks Codewarrior 9.3.
